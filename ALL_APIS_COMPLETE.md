## 🔧 Troubleshooting

### If Something Doesn't Work

**AI Features**:

- Verify API keys are correct
- Try switching AI provider
- Check API quota (unlikely to hit)

**Food Scanner**:
- Use good lighting
- Take clear photos
- Try different angles
- Switch between Gemini/Claude

**Exercise Library**:
- Check internet connection
- Try different search terms
- Use filters (body part, equipment)

**Nutrition Database**:
- Check internet connection
- Try simpler search terms
- Be specific with quantities
