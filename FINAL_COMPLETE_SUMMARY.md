## 🎓 Quick Start Guide

### 1. Test Without API Keys (2 minutes)
- Open app
- Go to Settings → My Custom Plans
- Create a workout or diet plan
- See real human exercise GIFs
- View quote of the day

### 2. Add API Keys (20 minutes)
1. **ExerciseDB** (5 min)
   - Get key at RapidAPI
   - Add to `exercise_api_service.dart`

2. **Nutritionix** (5 min)
   - Get credentials at Nutritionix
   - Add to `nutrition_api_service.dart`

3. **Groq AI** (5 min) ⭐ NEW
   - Get key at console.groq.com
   - Add to `groq_ai_service.dart`

4. **Gemini AI** (5 min) ⭐ NEW
   - Get key at makersuite.google.com
   - Add to `gemini_ai_service.dart`

### 3. Enjoy All Features!

- Chat with AI coach
- Generate AI workout plans
- Generate AI diet plans
- Create custom plans
- Track progress
